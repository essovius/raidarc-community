# RaidARC Community

Welcome to the **RaidARC Community**! This is the hub for community-contributed links and categories for [RaidARC.com](https://raidarc.com).

## How to Suggest

The easiest way to contribute is by opening an issue using one of the templates below:

- [**Suggest a New Link**](../../issues/new?template=new-link.yml) - Share a useful resource (wikis, guides, tools, streamers)
- [**Suggest a New Category**](../../issues/new?template=new-category.yml) - Propose a new category for organizing content

## Direct Contribution

If you are comfortable with GitHub, you can directly edit the data files and submit a Pull Request:

1. Fork this repository
2. Edit `data/links.json` or `data/categories.json`
3. Submit a pull request

All submissions are automatically validated and manually reviewed by maintainers before being synced to the site.

## Guidelines

- **Links**: Must be related to ARC Raiders and provide value to the community.
- **Categories**: Should reflect a distinct type of resource not covered by existing categories.

Be respectful and constructive. We're all here to make the ARC Raiders community better!
